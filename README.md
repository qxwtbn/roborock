**Quick FAQ**

**Q: How often do I need to empty the dustbin?**  
Depends on the model, but auto-empty docks can hold 7-10 weeks of dust. The robot itself needs emptying every few cleaning sessions if you don't have auto-empty.

**Q: Can it handle pet hair?**  
Yes, especially the newer models with the DuoDivide anti-tangle brush. But you'll still need to clean the brushes occasionally.

**Q: Does it work on dark floors?**  
Yes, the newer models with advanced sensors handle dark floors better than older LiDAR-only systems.

**Q: How loud is it?**  
About as loud as a regular vacuum when on max suction. You can schedule it to run when you're out.

**Q: Is the mopping function actually good?**  
It's more "maintenance mopping" than deep cleaning. Great for daily dust and light spills, not for sticky messes or serious grime.

**Q: Can I use it upstairs and downstairs?**  
Yes, newer models can save multiple floor maps. You just carry it to different floors.
